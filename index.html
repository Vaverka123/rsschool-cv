Code Sample

[Vowel Count](https://www.codewars.com/kata/54ff3102c1bad923760001f3): Return the number (count) of vowels in the given string. We will consider a, e, i, o, u as vowels for this Kata (but not y). The input string will only consist of lower case letters and/or spaces. JavaScript:

                    function getCount(str) {
                        const vowels = ['a', 'e', 'i', 'o', 'u'];
                        let count = 0;
                    
                        for (let char of str) {
                        if (vowels.includes(char)) {
                            count++;
                        }
                        }
                    
                        return count;
                    }
